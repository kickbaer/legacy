Training Registration

https://www.apac.redhat.com/training/courses

  • Select a course.
  • Click on Enroll Now.
  • Select a schedule.
  • Scroll down the t&c and accept
  • Fill out the registration form and submit it.
  • The data should be stored in the database.
  • An email should be sent out to the email address of the customer.
  • An email should be sent out to Red Hat with the booking information.

sync.php

https://www.apac.redhat.com/ADMIN/training/sync.php

  • This script serves the registration information for the course registrations. A script from an internal server (webqa.str.redhat.com) pulls these information and sends out the confirmation emails with the details for Red Hat. Slows your browser down by testing it.
  • an $id=INT in the url should filter the registrations from this id to the newest id. E.G: https://www.apac.redhat.com/ADMIN/training/sync.php?id=7233
