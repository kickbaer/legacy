User

https://www.cn.redhat.com/ADMIN/user/

  • Lists the names and other information of the user of the ADMIN SUITE.

clicking on Add user or edit leads to:
https://www.cn.redhat.com/ADMIN/user/edit.php3

  • All available formula data should be pre-filled when editing
  • After editing, the changes should be stored in the database.
    (check https://www.cn.redhat.com/ADMIN/user/)
  • When adding a new entry, the entry should be stored in the database.
    (check https://www.cn.redhat.com/ADMIN/user/)

By clicking on delete leads to:
https://www.cn.redhat.com/ADMIN/user/delete.php3

  • Lists the name and other information of the specified user.
  • Clicking on yes deletes the news entry.
    (check https://www.cn.redhat.com/ADMIN/user/)
